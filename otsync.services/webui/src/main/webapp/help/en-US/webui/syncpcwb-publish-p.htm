Publishing an Item or Folder

To publish an item or folder:

  1. Point to the itemâ€™s function menu and then click Publish.

    Contact your administrator if this option is not available.

    Standard users and external users cannot publish items to Content Server.

  2. If you are publishing an item, click one of the following:

    Publish a New Version

    Add the item as a new version of an existing item in Content Server.

    Publish to a folder

    Add the item as a new item to a Content Server location such as your Personal Workspace, a Folder, or a Project.

  3. Click Browse Content Server, and select one of the following:

    • A Content Server item, if you are publishing a new version of an item.

    • A Content Server folder, if you are a publishing a new item or a folder.

  4. Click one of the following:

    Retain a copy of your file

    Keep the existing item or folder in its present location in your Tempo Box folder.

    Move file into Content Server and delete original

    Delete the item or folder from your Tempo Box folder.

     

    Note:
    If you are publishing a shared folder owned by another user, you must retain a copy of the folder in your Tempo Box folder.

     

  5. Click Publish.
