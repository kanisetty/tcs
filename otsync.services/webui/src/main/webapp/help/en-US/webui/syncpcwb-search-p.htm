Searching for an Item or Folder

To search for an item or folder:

  1. Open the folder you want to search in.

  2. In the Search box in the middle pane, type the text you are searching for, and then click the magnifying glass .

     

    Note:
    You can only search for item and folder names. Depending on how your system is configured, you might be able to use the asterisk (*) as a wildcard in your search terms.

     

  3. Perform one or more actions in the following table.

    To Do this
    Open a result in the result list Click the file or folder name.
    Open the folder that contains a result Click the folder name next to the Found In link.
    Display the top-level view of your Tempo Box folder In the left pane, click Return to All Files.
