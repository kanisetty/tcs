Creating a Folder

To create a folder:

  1. Navigate to the location where you want to create a folder.

    You cannot create a folder in a folder for which you have read-only access. Read-only folders are identified by the lock overlay. For example, or .

  2. In the middle pane, click New Folder.

  3. Type a folder name in the box and then click Create.
