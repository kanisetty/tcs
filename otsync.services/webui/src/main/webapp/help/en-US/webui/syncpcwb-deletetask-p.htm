Deleting a Task

To delete a task:

  1. Select the folder containing a task that you created. You can only delete the tasks you created.

     

    Tip:
    Folders with active tasks will show a task icon . For example, . The task icon disappears when the status of all of the tasks is set to Completed or Cancelled, or all of the tasks in the folder are deleted.

     

  2. Click TASKS in the right pane. A list of tasks appears.

  3. Locate the task of interest and click the X in the top right corner of the task.
