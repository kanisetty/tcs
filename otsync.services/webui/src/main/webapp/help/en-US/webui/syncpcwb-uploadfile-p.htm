Adding an Item

To add an item:

  1. Navigate to the location where you want to add an item.

    You cannot add an item in a folder for which you have read-only access. Read-only folders are identified by the lock overlay. For example, or .

    You cannot add an item in a folder that has mandatory categories and attributes through Tempo Box. To determine if a folder that has mandatory categories and attributes, see Viewing Item or Folder History.

  2. In the middle pane, click Add File.

    Tempo Box only handles files up to 2 GB. Files larger than this will fail.

     

    Important:
    If you add content in a folder for which you have read/write access, the changes are synchronized to all users who have access to the folder. Read/write folders are identified by or .

    If you edit content in an Enterprise folder, the changes are synchronized to the folder in the Enterprise Workspace. Enterprise folders are identified as .

     

  3. Do one of the following:

    • If a Browse button appears, click Browse, select an item, and then click Upload.

    • If the File Upload dialog box appears, select an item, and then click Open.

 

Tip:
If you are using the latest supported version of Chrome, you can also add a file by dragging it from your desktop to the Web client browser window. You can only drag files, not folders. Dragging a file always copies it to the Web client, even if the browser indicates that the file is being moved.
