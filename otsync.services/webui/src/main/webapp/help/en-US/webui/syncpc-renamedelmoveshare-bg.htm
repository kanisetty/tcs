Moving, Renaming, and Deleting Shared Folders

The following conditions apply for shared folders that you own:

  • If you move a shared folder to another location in your Tempo Box folder, it remains shared for all other collaborators.

  • If you move a shared folder to a location outside of your Tempo Box folder, it is no longer shared, and other collaborators cannot see it. The folder becomes a normal folder and the content is deleted from your Tempo Box folder.

  • If you rename a shared folder, it is renamed for all other collaborators.

  • If you delete a shared folder, it is deleted for all other collaborators.

The following conditions apply for shared folders that you have permission to access, but do not own:

  • You cannot move the root of a shared folder, Enterprise folder, or Community file library.

  • If you move a shared folder outside of your user name folder, the folder is no longer shared with you. The folder becomes a normal folder and will no longer sync. If you accidentally move the folder outside of your user name folder, you must contact the folder owner to send you a new sharing invitation.

  • You cannot rename the root of a shared folder.

  • If you delete a shared folder, it is only removed from your Tempo Box folder. Other collaborators can continue to access the folder. If you accidentally deleted the folder, you must contact the folder owner to send you a new sharing invitation.

  • If you delete an Enterprise folder, it is only removed from your Tempo Box folder. Other collaborators can continue to access the folder. If a Content Server user accidentally deleted an Enterprise folder, they can add the folder again from the folder location in Content Server. For more information on adding Enterprise folders to your Tempo Box folder, see the OpenText Content Server User Help - Using OpenText Tempo Box (SYNCLLES-H-UGD).

  • You cannot rename the root of an Enterprise folder unless you have Edit Permissions permission to the folder.

  • You cannot rename or move the root of a file library in a Tempo Social community.
