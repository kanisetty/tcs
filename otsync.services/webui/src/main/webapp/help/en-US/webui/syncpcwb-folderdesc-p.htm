Adding and Editing a Folder Description

To add a folder description:

  1. Select a folder.

  2. Click DESCRIPTION in the right pane. If you are viewing a Content Server folder, the description from Content Server will display.

  3. Click the description box and enter or edit the description. You can add or edit the description only if you have read/write permissions to the folder.

  4. Click Update.
