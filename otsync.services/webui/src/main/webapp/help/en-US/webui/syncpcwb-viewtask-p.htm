Viewing a Task

To view a task:

  1. In the top, right pane, click the notification icon and then click the task notification. The folder containing the task opens.

     

    Tip:
    You can also view a task by clicking the folder which contains the task. Folders that contain active tasks will show the task icon . For example, .

     

  2. Click TASKS in the right pane. The list of all available tasks displays.
