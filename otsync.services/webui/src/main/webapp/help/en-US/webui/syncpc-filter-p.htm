Filtering the Folder View

To filter the folder view:

  • Click one of the following links in the left pane:

    • All Files – Shows all of the content in your Tempo Box folder.

    • My Files – Shows all of the files and folders that you own.

    • Shared to Me – Shows only the folders that have been shared with you.

    • Enterprise Content – Shows the Enterprise folders for which you have subscribed.

    • My Communities – Shows the file libraries in the Tempo Social communities that you follow.

    The My Files, Shared to Me, Enterprise Content, and My Communities filters only appear if you have at least one folder that meets the filter’s description.
