Viewing Item or Folder History

To view item or folder history:

  1. Point to the item’s function menu and then click History.

  2. If you are viewing an item’s history, click one of the following tabs:

    Audit History

    Displays each action undertaken on the item.

    Version History

    Displays a list of all previous versions of the item. To view a previous version, click the Document Name. Folders are not versionable and only display Audit History.

     

    Note:
    The contents of the Audit History tab depend on which event types your administrator has enabled for auditing.
