Responding to a Sharing Invitation

To respond to a sharing invitation:

  1. In the top, right pane, click the notification icon .

  2. Click the sharing invitation and do one of the following:

    • Click to accept the invitation.

      The shared folder is placed in the root of your Tempo Box folder and will appear as if you have read/write access, or as if you have read-only access. The folder name is appended with the user name of the folder owner. For example, Marketing Materials (jsmith).

      If you accepted a share invitation for an Enterprise object, the object is placed in the root of your Tempo Box folder and will appear as if you have read/write access or if you have read-only access. Depending on your permissions, the Enterprise object may append the parent object name to the parent object. For example, Payroll (Human Resources).

    • Click and then click Decline to decline the invitation.
