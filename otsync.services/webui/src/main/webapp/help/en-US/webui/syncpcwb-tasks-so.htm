Creating and Managing Tasks

Within a folder, you can create a task for yourself or assign a task to another colleague in a shared or Enterprise folder. For example, you can add a document in a shared folder and create a task for a collaborator to review the document. When you create a task, a notification is sent to the task recipient outlining the task description and due date, if provided. The notification icon appears in the top, right pane when a task is assigned to you. If enabled, you can also receive an email notification when you are assigned a task. For details, see Enabling Email Notifications.

 

Note:
The task notification will always appear regardless of the task status or even if the task is deleted.

 

Folders that contain active tasks will show the task icon . For example, . The task icon disappears when the status of all of the tasks is set to Completed or Cancelled, or all of the tasks in the folder are deleted. The task icon only shows on the folders which have tasks created against them. A folder with tasks created within a subfolder will not show the task icon on the parent folder.
