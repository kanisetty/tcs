Deleting a Shared Folder

To delete a shared folder that you own:

  1. Navigate to the target folder, identified as .

  2. Point to the item’s function menu and then click Delete. If Delete is not available, you have selected a folder that is shared with you, an Enterprise folder, or a file library from a Tempo Social Community. For more information, see Removing a Folder That is Shared With You, Removing an Enterprise Folder or Removing a File Library from a Tempo Social Community.

     

    Caution
    This will delete the folder from your Tempo Box folder, which means that the folder is deleted from all of your other OpenText Tempo Box clients such as your desktop clients, smartphones and tablets. This will also remove the folder for all collaborators of this folder. As an alternative, you can remove collaborators. For details, see Removing Collaborators from a Shared Folder.

     

  3. Click Delete.

To remove a folder that is shared with you:

  1. In the root of your Tempo Box folder, locate the target shared folder. Folders that are shared with you append the folder owner’s user name in the folder name. For example, Marketing Materials (jsmith).

  2. Point to the item’s function menu and then click Remove.

     

    Caution
    You must remove the actual folder and not delete the content inside the shared folder. Deleting the content in the shared folder will delete the content for all collaborators including the owner of the shared folder.

     

  3. Click Remove.

    This will sever the connection to the shared folder and remove it from your Tempo Box folder. This does not delete the folder for the owner or other collaborators.

To remove an Enterprise folder:

  1. In the root of your Tempo Box folder, locate the target Enterprise folder. Enterprise folders are identified as or .

  2. Point to the item’s function menu and then click Remove.

     

    Caution
    You must remove the actual folder and not delete the content inside the Enterprise folder. Deleting the content in the Enterprise folder will delete the content for all collaborators who have subscribed to the folder, as well as delete the content from the folder in the Enterprise Workspace.

     

  3. Click Remove.

    This will sever the connection to the folder and remove it from your Tempo Box folder. This does not delete the folder in the Enterprise Workspace. If you accidentally deleted the Enterprise folder, Content Server users can navigate to the folder in the Enterprise Workspace and add the folder once again. For more information on adding Enterprise folders to your Tempo Box folder, see OpenText Content Server User Help - Using OpenText Tempo Box (SYNCLLES-H-UGD).

To remove a file library from a Tempo Social community:

  1. In the root of your Tempo Box folder, locate the target folder. Community folders are identified as or .

  2. Point to the item’s function menu and then click Remove.

  3. Click Remove.

    This will sever the connection to the file library and remove it from your Tempo Box folder. This does not delete the file library in Tempo Social. If you accidentally deleted the file library, navigate to the community in Tempo Social, leave and then rejoin the community. For more information about Tempo Social communities, see the OpenText Tempo Social - Social Workplace Site User Help (WCSOC-H-UWS).
