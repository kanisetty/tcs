Renaming a File or Folder

To rename an item or folder:

  1. Point to the item’s function menu and then click Rename.

    When renaming a shared folder, the results vary depending on whether you are the shared folder’s owner. For more information, see Moving, Renaming, and Deleting Shared Folders.
