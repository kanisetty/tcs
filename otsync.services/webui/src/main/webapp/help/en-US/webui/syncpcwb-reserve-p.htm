Reserving and Unreserving a File

To reserve a file:

  1. Point to the item’s function menu and then click Reserve.

    You cannot reserve a file in a folder for which you have read-only access. Read-only folders are identified by the lock overlay. For example, or .

    If you are trying to reserve a file in an Enterprise folder for which you have read/write access and the Reserve option is not available, you do not have Reserve permission to the folder in Content Server. Contact your administrator to request this permission.

  2. Click Reserve. The file is marked with the overlay. For example, .

To unreserve a file:

  1. Point to the file’s function menu and then click Unreserve. If Unreserve is not available, the file is reserved by another user. To determine who reserved the file, you can check the audit history of the file. For details, see Viewing Item or Folder History. Contact the user and ask them to unreserve the file.

  2. Click Unreserve.
